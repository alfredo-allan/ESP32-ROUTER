    1.1 Objetivo do Projeto

    Transformar um ESP32-WROOM-32U em um roteador Wi-Fi capaz de:

    Conectar em uma rede existente
    Criar uma nova rede Wi-Fi
    Distribuir internet aos clientes
    Fazer NAT (Network Address Translation)
    Fornecer DHCP
    Realizar Port Forwarding
    Aplicar Firewall ACL
    Operar VPN WireGuard
    Permitir administração via Web

    1.3 Hardware Utilizado

    ESP32-WROOM-32U

    Característica Especificação
    Processador Dual Core Xtensa LX6
    Frequência 240 MHz
    Wi-Fi 2.4 GHz 802.11 b/g/n
    Bluetooth BLE 4.2
    Flash 4 MB
    Antena Externa U.FL

    1.4 Firmware Utilizado

    Projeto: esp32_nat_router

    Função: Roteador NAT completo

    Recursos: NAT, DHCP, Firewall, VPN, Port Forward, Web Interface, Console CLI

    1.5 Instalação do Firmware

    Apagar Flash

    esptool --chip esp32 --port /dev/ttyUSB0 erase-flash

    Gravar Firmware

    IMPORTANTE: Os offsets corretos foram: 
    esptool --chip esp32 \
--port /dev/ttyUSB0 \
--before default_reset \
--after hard_reset write-flash \
-z --flash_mode dio --flash_freq 40m --flash_size detect \
0x1000 firmware_esp32/bootloader.bin \
0x8000 firmware_esp32/partition-table.bin \
0xf000 firmware_esp32/ota_data_initial.bin \
0x20000 firmware_esp32/esp32_nat_router.bin

    2.1 Configuração Inicial

    Configurar Internet

    set_sta "Sousa 2" "061102dhi"

    Significado: STA = Station — Rede que o ESP32 vai acessar

    Configurar AP

    set_ap "KaliSonic" "drone2025"

    Significado: AP = Access Point — Rede criada pelo ESP32

    Reiniciar

    restart

    2.2 Verificação

    Status

    show status

    Exemplo de saída:

    Uplink AP: Sousa 2
Uplink IP: 10.0.0.103

AP Interface: enabled
Connected Clients: 1

    Configuração

    show config

    Exibe: SSID STA, Senha STA, SSID AP, Senha AP, IP AP, DNS, Segurança

    2.3 Scanner Wi-Fi

    Escanear redes:

    scan

    Mostra: SSID, RSSI, Canal, Segurança

    2.4 Alterações de Rede

    Alterar Nome da Rede

    set_ap "MinhaRede" "Senha123"

    Alterar Rede de Internet

    set_sta "NovaRede" "NovaSenha"

    Alterar IP do AP

    set_ap_ip 192.168.10.1

    Padrão: 192.168.4.1

    Alterar DNS

    set_ap_dns 8.8.8.8
    set_ap_dns 1.1.1.1

    2.5 NAT

    Verificar:

    set_ap_nat

    Ativar / Desativar:

    set_ap_nat on
    set_ap_nat off
    Normalmente: ON

    2.6 Potência de Transmissão

    Consultar:

    show config

    Alterar:

    set_tx_power 20

    Faixa: 2 a 20 dBm

    2.7 Ocultar Rede

    Ocultar SSID:

    set_ap_hidden on

    Mostrar:

    set_ap_hidden off

    2.8 Segurança Wi-Fi 

    set_ap_auth wpa2
    set_ap_auth wpa3
    set_ap_auth wpa2wpa3
    Modo Descrição
    wpa2 WPA2 padrão
    wpa3 WPA3 mais seguro
    wpa2wpa3 Modo híbrido

    2.9 Controle de MAC

    Bloquear cliente:

    dhcp_reserve block AA:BB:CC:DD:EE:FF

    Reservar IP:

    dhcp_reserve add AA:BB:CC:DD:EE:FF 192.168.4.100

    2.10 Port Forward

    Exemplo:

    portmap add TCP 8080 192.168.4.10 80

    Resultado:

    IP externo:8080
       ↓
192.168.4.10:80

    2.11 Firewall ACL

    Listas disponíveis: to_esp, from_esp, to_ap, from_ap

    Permitir:

    acl to_ap TCP any any allow

    Negar:

    acl to_ap TCP any any deny

    2.12 Web Interface

    Verificar:

    web_ui

    Ativar:

    web_ui enable

    Porta:

    web_ui port 80

    2.13 Console Remoto

    Ativar:

    remote_console enable

    Porta: 2323

    Conectar:

    telnet 192.168.4.1 2323

    2.14 WireGuard VPN

    Configurar:

    set_vpn

    Permite: Tunelamento, Acesso remoto, Proteção de tráfego

    2.15 Estatísticas

    Ver tráfego:

    bytes

    Resetar:

    bytes reset

    2.16 Reinício e Reset de Fábrica

    Reinício:

    restart
    Reset de Fábrica — Apaga tudo: 
    factory_reset

    3.2 Limitações do ESP32

    O ESP32 NÃO é um roteador profissional.

    Limites práticos:

    • 2.4 GHz apenas
    • 10~20 clientes simultâneos
    • Throughput limitado
    • CPU compartilhada com NAT

    Velocidade típica: 10 a 30 Mbps — Dependendo do ambiente.
